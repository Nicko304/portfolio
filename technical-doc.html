Free 3rd Party Hosting

A free method to host a server via a third-party website portal.

  • Requires no money!
  • Not as good as the other options.
Typically capped with very low memory and processing power.
Free Desktop Hosting

A very good option if you are able to host it yourself!

  • Requires no money!
  • Hosting depends on your computer and ability to activate the server with it.
  • Comes with the most freedom to add whatever features, plugins, or mods that you desire

To run your server, you need a .bat file with the following format typed:

java -Xmx####M -Xms####M -jar FILE_NAME.jar nogui

Working example:

java -Xmx1024M -Xms1024M -jar vanilla-1.19.1.jar nogui
Paid 3rd Party Hosting

Similar to desktop hosting, but not free.

Usually requires monthly subscription and prices depend on memory needed.

1G = 1024M (can be used interchangably in the .bat file as 1G represents 1 gigabyte with this format)
Realm Hosting

Paid and stable hosting with official sources.

Costs money and can have limitations at times.

Free Multiplayer Servers

Free, fun, and so much variety!

Some servers have microtransactions and can limit your options or enjoyment.
